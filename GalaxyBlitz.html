Galaxy Blitz

Collect powerups, max stats, and enter Blitz!
Solo Project

This project was my individual game of a 3-game assignment
The main focus of this game is it's progression system in which the player can power up their ship to enter Blitz Mode.


Orbital Disco

Reach your dance partner!
Team Size : 6
Contributions: Movement mechanics, Sprite animations, Sfx

The objective is to reach your dance partner on the other side of the map without touching the enemies.
The game features a unique charged press and hold mechanic that propels the character further the longer the button is held down. With this mechanic, players can precisely control both the speed and direction of the character using a single button. The game also features a procedural spawn system that increases the number of enemies that spawn in with each completed level.


Humanity's Demise

Destroy the Humans!
Team Size : 2
Contributions: Gameplay Programming

Classic Galaga style gameplay where the the humans grow stronger as you progress.
This game's code is based on Galaxy Blitz but using mouse movement instead of single click.
