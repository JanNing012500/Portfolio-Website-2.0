Raveball Arena

Overview

  • Engine: UE5
  • Roles: Project Manager, UI/UX Designer, Blueprints Programmer
  • Project date: Senior Capstone Project
  • Project URL: RaveballArena.com

    • Competitive high powered rhythmic soccer game made using Unreal Engine 5.


Details

As the project manager for my Senior Capstone Project, I led a team in the development of a competitive third-person shooter soccer game called Raveball Arena. I utilized Azure DevOps for version control and sprint planning, holding weekly meetings to ensure that all team members were on track with their assigned tasks. Additionally, I was responsible for the complete game UI and HUD, as well as contributing to gameplay programming.

Before the official release of Unreal Engine 5, we successfully built it from source and created a basic prototype of our game during our winter break. We presented the prototype to professors and classmates, successfully recruiting additional team members for 3D assets and sound design.
