Orbital Disco

Overview

  • Language: Javascript
  • Team Size: 6
  • Roles: Project Manager, Programmer
  • Project date: December 2021
  • Project Link: Play

    • Group 2D Endless JavaScript game made with a lightweight JS library (Kenta Cho's crisp-game-lib). Focuses on single-button input and simple graphics primitives to narrow design parameters. The objective of Orbital Disco is to reach your dance partner on the other side of the map without touching the enemies.


Details

Contributions: Movement mechanics, Sprite animations, Sfx

Orbital Disco is a game that challenges players to reach their dance partner on the other side of the map without touching the enemies. The game features a unique charged press and hold mechanic that propels the character further the longer the button is held down. The direction the character is launched towards is based on the position of an orbiting clockwise indicator that surrounds the player. With these two factors combined, players can precisely control both the speed and direction of the character using a single button. However, this mechanic adds a layer of difficulty when controlling the movements because the longer the button is held, the further and faster the character will launch, leaving players with few chances to let go of the button at the right time when determining the direction. The game also features a procedural spawn system that increases the number of enemies that spawn in with each completed level. For this project, I developed the movement mechanics, sprite animations, and all the sound effects. Overall, Orbital Disco is a challenging and innovative game that requires players to think critically and react quickly in order to reach their dance partner.

This project was the group project of a 3-game assignment where students needed to make : one game individually (Galaxy Blitz), one game in collaboration with one other collective member (Humanity's Demise), and one game in collaboration with two or more collective members (Orbital Disco).
