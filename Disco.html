Details

Contributions: Movement mechanics, Sprite animations, Sfx

Orbital Disco is a challenging 2D endless game made with a lightweight JS library. The objective is to reach your dance partner on the other side of the map without touching the enemies. The game features a unique charged press and hold mechanic that propels the character further the longer the button is held down. With this mechanic, players can precisely control both the speed and direction of the character using a single button. The game also features a procedural spawn system that increases the number of enemies that spawn in with each completed level.

This project was the group project of a 3-game assignment where students needed to make : one game individually (Galaxy Blitz), one game in collaboration with one other collective member (Humanity's Demise), and one game in collaboration with two or more collective members (Orbital Disco).

This game is playable in browser on Mobile or PC
