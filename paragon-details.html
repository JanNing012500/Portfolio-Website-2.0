Paragon UI Project

Overview

  • Solo Project
  • Engine: UE5
  • Project Date: December 2022

    • Paragon UI features a character selection screen with customizable attributes for three unique characters, each with their own set of skills and attributes. The interface allows players to adjust statistics and even customize the characters' names and color themes.


Summary

As part of my ongoing professional development, I completed a self-directed learning project on creating a video game User Interface for character customization using Unreal Engine which featured 3 unique characters with various customizable options. I took the initiative to find resources and complete this project in my free time, which allowed me to gain a deeper understanding of the ins-and-outs of character UI, which I will implement towards future projects.



Breakdown

I implemented the use of Unreal Engine's Widget Blueprints to create a character-based user interface for a game similar to that of Final Fantasy and God of War games. This UI allows the player to select their desired character and also customize many of their attributes such as toggling auto move, their speed, health regeneration rate and the color scheme. While designing the menus, I laid out a character menu selection screen and made it interactive by bounding data from a data table to elements in the character menu and created a game save and load system. I also made the character data editable and updated saved data, and toggled a selection of playable characters. I also imported assets and implemented various types of widgets, from buttons, images, text boxes, horizontal/vertical boxes, spinner boxes, sliders, and uniform grid panels. Through working on this project, I gained skills in creating variables, writing functions, creating scripts, animating widgets, and making UI alterations that affected in-game characters.
