Galaxy Blitz

Overview

  • Solo Project
  • Language: Javascript
  • Project date: December 2021
  • Project Link: Play

    • Individual 2D Endless JavaScript game made with a lightweight JS library (Kenta Cho's crisp-game-lib). Focuses on single-button input and simple graphics primitives to narrow design parameters. This main focus of this game is it's progression system in which the player can power up their ship to enter Blitz Mode.


Details

Galaxy Blitz was a solo project that I developed, featuring a side-to-side movement controlled by a single button. Players can change the direction of their character by clicking the button, or they can hold it down to stay in one spot. As players progress through the game and collect power-ups, the size of their bullets increases along with the rate of fire. Once players collect enough power-ups and reach the maximum fire rate and bullet size, they enter Blitz Time, which gives them a score multiplier and an extra life. If players lose the extra life, Blitz Time ends. After some initial playtesting, I added a mechanic that triggers when players lose a life - the full screen map-clear. This mechanic helps to give players a few split seconds to reposition themselves after losing a life, as the speed of the game progressively increases. Overall, Galaxy Blitz is a fast-paced and challenging game that requires players to think and react quickly in order to survive and achieve high scores.

This project was my individual game of a 3-game assignment where students needed to make : one game individually, one game in collaboration with one other collective member (Humanity's Demise), and one game in collaboration with two or more collective members (Orbital Disco).
